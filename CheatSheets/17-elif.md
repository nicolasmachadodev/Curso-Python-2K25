# Cheatsheet: elif

- elif: "else if", permite testar múltiplas condições.
- Usado entre if e else.
- Sintaxe:
  if cond1:
      # código
  elif cond2:
      # código
  else:
      # código
- Exemplo:
  if nota >= 7:
      print("Aprovado")
  elif nota >= 5:
      print("Recuperação")
  else:
      print("Reprovado")
