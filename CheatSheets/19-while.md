# Cheatsheet: while

- while: laço de repetição baseado em condição.
- Executa o bloco enquanto a condição for verdadeira.
- Sintaxe:
  while condicao:
      # código
- Cuidado com loops infinitos!
- Pode usar break e continue.
- Exemplo:
  contador = 0
  while contador < 5:
      print(contador)
      contador += 1
