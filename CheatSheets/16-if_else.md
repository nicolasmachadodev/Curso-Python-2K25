# Cheatsheet: if, else

- if: executa um bloco de código se a condição for verdadeira.
- else: executa se a condição do if for falsa.
- Sintaxe:
  if condicao:
      # código
  else:
      # código
- Pode usar operadores de comparação (==, !=, >, <, >=, <=) e lógicos (and, or, not).
- Exemplo:
  if idade >= 18:
      print("Maior de idade")
  else:
      print("Menor de idade")
